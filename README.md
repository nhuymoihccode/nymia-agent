# NYMIA RETREAT - Internal Smart Assistant (AI Agent)

An AI-powered internal agent built during Kaggle's 5-Day AI Agents Intensive Course. This agent helps resort staff manage room availability and securely inspect revenue metrics.

## Key Course Concepts Applied
1. **Agent Skills (Antigravity CLI/IDE):** Powered by Gemini 3.5 Flash, structured using local system instructions.
2. **Data Integration:** Dynamically interfaces with a local `data.json` storage database.
3. **Security Guardrails:** Protects sensitive business data by challenging users for a secure PIN before exposing revenue.

## Local Setup Instructions
1. Download all files from this repository into a folder named `nymia-agent`.
2. Open this folder using **Antigravity IDE**.
3. The Agent will automatically detect `SKILL.md` and `data.json`. You can immediately interact with the agent in the right-side chat panel.
